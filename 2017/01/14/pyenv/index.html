Pyenv and Virtualenv

Posted on 2017-01-14 | In Python | |

Virtualenv is useful when you have multiple packages dependency demands. But is’s very tedious to deal with different python versions.

As a complement, Pyenv is a perfect tool to help you control different versions of python, like python2, python3, anaconda.

Personally, I use ‘pyenv’ and its plug-in ‘pyenv-virtualenv’ to manage the python envoronment.

Install Pyenv

General Approach

  • Install pyenv at /Home/.pyenv

  • Define environment variable PYENV_ROOT to point to the path of pyenv

  • Add path for command line operation

  • Add pyenv init to your shell to enable shims and autocompletion.

    • make sure eval “$(pyenv init -)” is at the end of the shell configuration file
For Ubuntu

Ubuntu note: Modify your ~/.zshrc file instead of ~/.zsh_profile.
